American Sign Language (ASL) is the primary language used by many deaf individuals in North America, and it is also used by hard-of-hearing and hearing individuals. The language is as rich as spoken languages and employs signs made with the hand, along with facial gestures and bodily postures.

Create Gesture

Capture dynamic hand signs with precision using the 'Create Gesture' function, providing valuable data for ML training and accurate predictions.

  • Select the Hand from the list.
  • Enter the Gesture Name.
  • Click on Start Capture button to start the process.
  • Click on Stop Capture button to stop the process.

Create Dataset

Compile images captured for each gesture, forming a comprehensive dataset that will serve as the foundation for training machine learning models.

  • Click the Start button to start the process.
  • Click the Stop button to stop the process.

No Gesture found inside the Data folder.

Train Model

Utilizing the dataset created to train selected machine learning models, which will subsequently be used for prediction purposes.

  • Select the preferred Machine Learning Model from the list.
  • Click on the Confirm button to begin the training process.

Scan Gesture

The trained machine learning models are deployed to analyze and recognize hand gestures showcased in the live webcam feed, making predictions based on the learned patterns and providing real-time gesture recognition.

  • Select the Preferred Model from the list.
  • Click Start button to start the recognition process.
  • Click Save button to save the recognised sentence.
  • Click Read button to hear the recognised sentence.
  • Click Stop button to stop the recognition process.
